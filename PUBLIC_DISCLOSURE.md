# Public Disclosure

This repository is intentionally public and intentionally limited.

## Public Material

The following material is safe to keep here:

- High-level project overview
- General research direction
- Non-sensitive architecture notes
- Documentation and collaboration guidelines
- Public-facing descriptions of dashboards, alerts, and testing workflows
- Non-sensitive screenshots or summaries when approved for release

## Private Material

The following material must remain outside this public repository:

- Full Expert Advisor source code
- Exact entry, exit, scoring, and filtering rules
- Exact risk sizing formulas
- Broker details, account identifiers, and platform-specific credentials
- API keys, Telegram bot tokens, chat identifiers, and secret configuration
- Raw Strategy Tester exports, optimization files, private notes, and proprietary research logs

## Safety Rule

If a file contains sensitive trading logic, credentials, account details, private testing data, or anything that could expose the research edge, it should not be committed here.

This repository exists to explain the project professionally, not to publish the private implementation.
