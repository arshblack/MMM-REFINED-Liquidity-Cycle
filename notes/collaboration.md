# Collaboration

This repository is open for safe collaboration around presentation, documentation, architecture discussion, and testing process.

## Good Contributions

- Improve public documentation clarity
- Suggest safer project organization
- Improve disclosure wording
- Propose non-sensitive testing templates
- Review dashboard presentation at a high level
- Help with public README and notes quality

## Not Accepted Publicly

- Full EA source code
- Exact entry or exit logic
- Account details or broker exports
- Telegram tokens, API keys, or chat identifiers
- Raw backtest files or private optimization results
- Proprietary research notes

## Working Rule

When in doubt, keep it private first and document only the high-level concept publicly.
